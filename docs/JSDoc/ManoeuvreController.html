Class: ManoeuvreController

ManoeuvreController

new ManoeuvreController()

Class that contstructs the tube and tube mesh from the array of manoeuvres parsed from user input. the ManoeuvreController#addTube method loops through the OLAN moves entererd, and breaks these down to their instructions, and creates an overall tube. This class is a subclass of AnimationController.
